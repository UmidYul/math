Тема первая
Возведение комплексного числа в степень. Извлечение корня. Показательная функция. Формула Эйлера

1.1 Тригонометрическая форма

Комплексное число $z = x + iy$ может быть записано в тригонометрической форме:

Определение
$$z = r(\cos\varphi + i\sin\varphi), \quad r = |z| = \sqrt{x^2 + y^2}, \quad \varphi = \arg z$$

Модуль $r$ задаёт удалённость числа от начала координат на комплексной плоскости, аргумент $\varphi$ — угол с положительным направлением вещественной оси.

1.2 Формула Эйлера

Формула Эйлера устанавливает фундаментальную связь между показательной и тригонометрической функциями:

Формула Эйлера
$$e^{i\varphi} = \cos\varphi + i\sin\varphi$$

Благодаря этому любое комплексное число записывается в показательной форме $z = re^{i\varphi}$, которая наиболее удобна при алгебраических операциях.

1.3 Возведение в степень. Формула Муавра

При возведении комплексного числа в целую степень $n$ модуль возводится в ту же степень, а аргумент умножается на $n$:

Формула Муавра
$$z^n = r^n(\cos n\varphi + i\sin n\varphi) = r^n e^{in\varphi}$$

Формула справедлива для любого целого $n$, в том числе отрицательного.

1.4 Извлечение корня $n$-й степени

Операция извлечения корня даёт в общем случае ровно $n$ различных значений:

Корень n-й степени
$$w_k = \sqrt[n]{r}\left(\cos\dfrac{\varphi + 2\pi k}{n} + i\sin\dfrac{\varphi + 2\pi k}{n}\right), \quad k = 0, 1, \ldots, n-1$$

Геометрический смысл: все $n$ корней располагаются в вершинах правильного $n$-угольника, вписанного в окружность радиуса $\sqrt[n]{r}$, равномерно разделяя её.

1.5 Показательная функция комплексного переменного

Для комплексного числа $z = x + iy$ показательная функция определяется как:

Определение
$$e^z = e^{x+iy} = e^x(\cos y + i\sin y)$$

Основные свойства: $|e^z| = e^x$; $\arg(e^z) = y$; $e^{z_1+z_2} = e^{z_1}\cdot e^{z_2}$; функция периодична с периодом $2\pi i$.

Тема вторая
Приложения рядов Фурье на $[0,\pi]$. Разложение по синусам и косинусам

2.1 Ряд Фурье на $[-\pi, \pi]$

Любая кусочно-непрерывная функция $f(x)$, заданная на $[-\pi, \pi]$, может быть разложена в ряд Фурье:

Ряд Фурье
$$f(x) = \dfrac{a_0}{2} + \sum_{n=1}^{\infty}(a_n \cos nx + b_n \sin nx)$$

Коэффициенты Фурье вычисляются по формулам:

Коэффициенты Фурье
$$a_n = \dfrac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\, dx, \qquad b_n = \dfrac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\, dx$$

2.2 Разложение по синусам на $[0,\pi]$

Если требуется разложить функцию $f(x)$, заданную лишь на $[0,\pi]$, только по синусам, следует продолжить её нечётным образом на $[-\pi, 0]$. Тогда $a_n = 0$ для всех $n$, и ряд принимает вид:

Разложение по синусам
$$f(x) = \sum_{n=1}^{\infty} b_n \sin nx, \qquad b_n = \dfrac{2}{\pi}\int_{0}^{\pi} f(x)\sin nx\, dx$$

2.3 Разложение по косинусам на $[0,\pi]$

При чётном продолжении функции на $[-\pi, 0]$ все коэффициенты $b_n = 0$, и получается разложение только по косинусам:

Разложение по косинусам
$$f(x) = \dfrac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos nx, \qquad a_n = \dfrac{2}{\pi}\int_{0}^{\pi} f(x)\cos nx\, dx$$

2.4 Теорема Дирихле (условие сходимости)

Теорема. Если $f(x)$ — кусочно-монотонная функция с конечным числом точек разрыва первого рода, то её ряд Фурье сходится к $f(x)$ в точках непрерывности и к $\dfrac{f(x^-)+f(x^+)}{2}$ в точках разрыва.
Ключевые формулы — Тема 2
Тип разложения Коэффициенты
По синусам и косинусам $[-\pi,\pi]$ $a_n = \frac{1}{\pi}\int_{-\pi}^{\pi}f\cos nx\,dx$, $\;b_n = \frac{1}{\pi}\int_{-\pi}^{\pi}f\sin nx\,dx$
По синусам $[0,\pi]$ $b_n = \dfrac{2}{\pi}\int_0^{\pi}f(x)\sin nx\,dx$
По косинусам $[0,\pi]$ $a_n = \dfrac{2}{\pi}\int_0^{\pi}f(x)\cos nx\,dx$

Тема третья
Движение тела по течению. Ускорение и его составляющие. Законы Кеплера

3.1 Векторное описание движения

Положение тела в пространстве задаётся радиус-вектором $\mathbf{r}(t)$. Скорость и ускорение определяются как производные:

Скорость и ускорение
$$\mathbf{v} = \dot{\mathbf{r}} = \dfrac{d\mathbf{r}}{dt}, \qquad \mathbf{a} = \dot{\mathbf{v}} = \ddot{\mathbf{r}} = \dfrac{d^2\mathbf{r}}{dt^2}$$

3.2 Разложение ускорения по естественным осям

Ускорение точки при криволинейном движении раскладывается на два компонента — касательный (тангенциальный) и нормальный (центростремительный):

Составляющие ускорения
$$\mathbf{a} = a_\tau\,\boldsymbol{\tau} + a_n\,\mathbf{n}$$ $$a_\tau = \dfrac{dv}{dt} = \dot{v}, \qquad a_n = \dfrac{v^2}{R}$$

Здесь $v = |\mathbf{v}|$ — скорость, $R$ — радиус кривизны траектории в данной точке, $\boldsymbol{\tau}$ — единичный касательный вектор, $\mathbf{n}$ — единичный вектор главной нормали.

Полное ускорение: $|\mathbf{a}| = \sqrt{a_\tau^2 + a_n^2}$.

3.3 Движение тела в поле тяготения

Тело, брошенное под углом к горизонту (или движущееся «по течению»), подчиняется уравнениям:

Уравнения движения (без сопротивления)
$$x(t) = v_0\cos\alpha \cdot t, \qquad y(t) = v_0\sin\alpha \cdot t - \dfrac{g t^2}{2}$$

Дальность полёта и время достижения максимума высоты:

$$L = \dfrac{v_0^2 \sin 2\alpha}{g}, \qquad t_{\max} = \dfrac{v_0\sin\alpha}{g}$$

3.4 Законы Кеплера

I закон (эллипса). Каждая планета движется по эллипсу, в одном из фокусов которого находится Солнце.
II закон (площадей). Радиус-вектор планеты за равные промежутки времени заметает равные площади: $$\dfrac{dS}{dt} = \dfrac{1}{2}\,|\mathbf{r} \times \mathbf{v}| = \text{const}$$
III закон (гармонический). Квадраты периодов обращения планет относятся как кубы больших полуосей их орбит: $$\dfrac{T_1^2}{T_2^2} = \dfrac{a_1^3}{a_2^3}$$
Ключевые формулы — Тема 3
Величина Формула
Тангенциальное ускорение $a_\tau = \dot{v}$
Нормальное ускорение $a_n = v^2/R$
Полное ускорение $|\mathbf{a}| = \sqrt{a_\tau^2 + a_n^2}$
Дальность полёта $L = v_0^2\sin2\alpha\,/\,g$
III закон Кеплера $T^2/a^3 = \mathrm{const}$

Тема четвёртая
Функции нескольких переменных. Частные производные. Экономические задачи

4.1 Основные понятия

Функция $z = f(x, y)$ ставит в соответствие каждой точке $(x, y)$ области $D \subset \mathbb{R}^2$ единственное вещественное число $z$. Область $D$ называется областью определения.

Полный дифференциал функции $f(x,y)$ в точке $(x_0, y_0)$:

Полный дифференциал
$$dz = \dfrac{\partial f}{\partial x}\,dx + \dfrac{\partial f}{\partial y}\,dy = f'_x\,dx + f'_y\,dy$$

4.2 Частные производные

Частная производная по $x$ вычисляется при фиксированном $y$, и наоборот:

Определение
$$f'_x = \dfrac{\partial f}{\partial x} = \lim_{\Delta x \to 0}\dfrac{f(x+\Delta x,\, y) - f(x,y)}{\Delta x}$$

Смешанные производные второго порядка при непрерывности совпадают: $f''_{xy} = f''_{yx}$.

4.3 Градиент и производная по направлению

Градиент
$$\nabla f = \mathrm{grad}\, f = \left(\dfrac{\partial f}{\partial x},\, \dfrac{\partial f}{\partial y}\right)$$

Производная функции по направлению единичного вектора $\mathbf{l} = (\cos\alpha, \cos\beta)$:

$$\dfrac{\partial f}{\partial \mathbf{l}} = \dfrac{\partial f}{\partial x}\cos\alpha + \dfrac{\partial f}{\partial y}\cos\beta$$

Градиент указывает направление наибольшего возрастания функции; его модуль равен наибольшему значению производной по направлению.

4.4 Применение в экономике

В экономических моделях функция $Q = f(K, L)$ описывает объём выпуска в зависимости от капитала $K$ и труда $L$ (производственная функция). Частные производные $\partial Q/\partial K$ и $\partial Q/\partial L$ называются предельными продуктами факторов производства.

Задачи на нахождение минимума затрат или максимума выпуска при ограничениях на ресурсы решаются методом множителей Лагранжа (см. Тему 5).

Ключевые формулы — Тема 4
Понятие Формула
Полный дифференциал $dz = f'_x\,dx + f'_y\,dy$
Градиент $\nabla f = (f'_x,\, f'_y)$
Производная по направлению $\partial f/\partial\mathbf{l} = f'_x\cos\alpha + f'_y\cos\beta$
Смешанная производная $f''_{xy} = f''_{yx}$ (при непрерывности)

Тема пятая
Условный и безусловный экстремум. Метод множителей Лагранжа

5.1 Безусловный экстремум

Необходимое условие экстремума: в точке экстремума все частные производные первого порядка обращаются в нуль. Точки, в которых выполнено это условие, называются стационарными:

Необходимое условие
$$f'_x(x_0, y_0) = 0, \qquad f'_y(x_0, y_0) = 0$$

Достаточное условие. Пусть $A = f''_{xx}$, $B = f''_{xy}$, $C = f''_{yy}$ вычислены в стационарной точке. Введём дискриминант $\Delta = AC - B^2$:

Достаточное условие
$$\Delta = AC - B^2 \begin{cases} > 0,\; A < 0 & \Rightarrow \text{максимум} \\> 0,\; A > 0 & \Rightarrow \text{минимум} \\ < 0 & \Rightarrow \text{седловая точка (не экстремум)} \\=0 & \Rightarrow \text{требуется дополнительное исследование} \end{cases}$$

5.2 Условный экстремум

Задача условного экстремума: найти экстремум функции $f(x,y)$ при наличии ограничения $\varphi(x,y) = 0$. Переменные $x$ и $y$ уже не являются независимыми.

5.3 Метод множителей Лагранжа

Вводится вспомогательная функция — функция Лагранжа:

Функция Лагранжа
$$\mathcal{L}(x, y, \lambda) = f(x, y) + \lambda\,\varphi(x, y)$$

Необходимые условия условного экстремума — система уравнений:

Система Лагранжа
$$\begin{cases} \mathcal{L}'_x = f'_x + \lambda\,\varphi'_x = 0 \\ \mathcal{L}'_y = f'_y + \lambda\,\varphi'_y = 0 \\ \varphi(x,y) = 0 \end{cases}$$

Решение системы даёт подозрительные на экстремум точки и значение множителя $\lambda$. Характер точки определяется из содержательного смысла задачи или с помощью дополнительного анализа.

Обобщение. При нескольких ограничениях $\varphi_k(x,y,\ldots) = 0$ вводится столько же множителей $\lambda_k$: $\;\mathcal{L} = f + \sum_k \lambda_k \varphi_k$.
Ключевые формулы — Тема 5
Понятие Формула
Стационарные точки $f'_x = 0,\; f'_y = 0$
Дискриминант $\Delta = f''_{xx}\cdot f''_{yy} - (f''_{xy})^2$
Функция Лагранжа $\mathcal{L} = f(x,y) + \lambda\,\varphi(x,y)$
Условия Лагранжа $\mathcal{L}'_x = 0,\; \mathcal{L}'_y = 0,\; \varphi = 0$

Тема шестая
Экстремум функции многих переменных. Наибольшее и наименьшее значения в замкнутой области

6.1 Экстремум функции нескольких переменных

Для функции $f(x_1, x_2, \ldots, x_n)$ стационарные точки находятся из системы:

Необходимое условие
$$\dfrac{\partial f}{\partial x_1} = 0, \quad \dfrac{\partial f}{\partial x_2} = 0, \quad \ldots, \quad \dfrac{\partial f}{\partial x_n} = 0$$

Достаточное условие экстремума в случае $n$ переменных формулируется через знакоопределённость матрицы вторых производных — матрицы Гессе (гессиана):

Матрица Гессе
$$H = \begin{pmatrix} f''_{x_1x_1} & f''_{x_1x_2} & \cdots \\ f''_{x_2x_1} & f''_{x_2x_2} & \cdots \\ \vdots & \vdots & \ddots \end{pmatrix}$$
Критерий Сильвестра. Если все угловые миноры $\Delta_k$ матрицы Гессе положительны ($\Delta_k > 0$), то точка является минимумом. Если знаки чередуются ($\Delta_k\!: -,+,-,\ldots$), то точка является максимумом.

6.2 Наибольшее и наименьшее значения в замкнутой области

В отличие от задачи на экстремум, требуется найти глобальный максимум и минимум функции на замкнутой ограниченной области $\overline{D}$. Алгоритм:

Алгоритм
\begin{align} &\textbf{1.}\;\text{Найти стационарные точки внутри } D,\text{ вычислить } f \text{ в них.}\\ &\textbf{2.}\;\text{Исследовать } f \text{ на границе } \partial D\text{ (задача условного экстремума).}\\ &\textbf{3.}\;\text{Сравнить все найденные значения; наибольшее и наименьшее — искомые.} \end{align}

6.3 Замечание об экономических приложениях

В экономике задача нахождения наибольшего значения функции прибыли $\Pi(x, y)$ или наименьшего значения функции затрат $C(x, y)$ на замкнутой области сводится к описанному алгоритму. Граница области в этом случае задаётся бюджетными ограничениями или технологическими пределами.

Ключевые формулы — Тема 6
Понятие Формула / условие
Стационарные точки $\nabla f = \mathbf{0}$, т.е. $f'_{x_k} = 0\;\forall k$
Матрица Гессе $H_{ij} = \partial^2 f / \partial x_i \partial x_j$
Минимум (Сильвестр) $\Delta_1 > 0,\;\Delta_2 > 0,\;\ldots > 0$
Максимум (Сильвестр) $\Delta_1 < 0,\;\Delta_2> 0,\;\Delta_3 < 0,\;\ldots$
Глобальный экстремум $\max_{\overline{D}} f = \max\!\left\{f\text{ внутри} \cup f\text{ на }\partial D\right\}$

Тема седьмая
Тройной интеграл. Вычисление тройных интегралов в декартовых координатах

7.1 Определение тройного интеграла

Пусть функция $f(x, y, z)$ определена и ограничена в замкнутой области $V \subset \mathbb{R}^3$. Разобьём $V$ на $n$ элементарных частей объёмом $\Delta V_k$, выберем в каждой произвольную точку $(\xi_k, \eta_k, \zeta_k)$ и составим интегральную сумму. Предел этой суммы при измельчении разбиения называется тройным интегралом:

Определение
$$\iiint_V f(x,y,z)\,dV = \lim_{\lambda \to 0} \sum_{k=1}^n f(\xi_k,\eta_k,\zeta_k)\,\Delta V_k$$

Геометрический смысл при $f \equiv 1$: $\iiint_V dV = \text{объём области } V$.

7.2 Сведение к повторному интегралу

Основной метод вычисления тройного интеграла в декартовых координатах — сведение к трём последовательным определённым интегралам. Если область $V$ описывается неравенствами:

$$a \leq x \leq b, \quad \varphi_1(x) \leq y \leq \varphi_2(x), \quad \psi_1(x,y) \leq z \leq \psi_2(x,y)$$

то тройной интеграл вычисляется как повторный:

Повторный интеграл
$$\iiint_V f\,dV = \int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} dy \int_{\psi_1(x,y)}^{\psi_2(x,y)} f(x,y,z)\,dz$$

Интегрирование ведётся изнутри наружу: сначала по $z$, затем по $y$, затем по $x$.

7.3 Замена переменных. Цилиндрические и сферические координаты

При замене переменных в тройном интеграле вводится якобиан преобразования $J$:

Замена переменных
$$\iiint_V f(x,y,z)\,dV = \iiint_{V'} f\bigl(x(u,v,w),\ldots\bigr)\,|J|\,du\,dv\,dw$$

Для цилиндрических координат $(r, \varphi, z)$: $x = r\cos\varphi$, $y = r\sin\varphi$, $z = z$; якобиан $J = r$.

Для сферических координат $(r, \theta, \varphi)$: $x = r\sin\theta\cos\varphi$, $y = r\sin\theta\sin\varphi$, $z = r\cos\theta$; якобиан $J = r^2\sin\theta$.

В сферических координатах
$$\iiint_V f\,dV = \iiint f\,r^2\sin\theta\,dr\,d\theta\,d\varphi$$
Ключевые формулы — Тема 7
Понятие Формула
Повторный интеграл $\int_a^b dx\int_{\varphi_1}^{\varphi_2} dy\int_{\psi_1}^{\psi_2} f\,dz$
Объём тела $V = \iiint_V dV$
Якобиан (цилиндр.) $J = r$
Якобиан (сферич.) $J = r^2\sin\theta$

Тема восьмая
Геометрические и физические приложения двойного интеграла. Объём тела. Площадь поверхности

8.1 Площадь плоской фигуры

Площадь плоской области $D$ выражается через двойной интеграл:

Площадь
$$S = \iint_D dA = \iint_D dx\,dy$$

8.2 Объём тела

Объём тела, ограниченного снизу плоскостью $z = 0$, сверху поверхностью $z = f(x,y) \geq 0$ над областью $D$:

Объём цилиндроида
$$V = \iint_D f(x,y)\,dx\,dy$$

8.3 Площадь поверхности

Если поверхность задана явно уравнением $z = f(x,y)$, её площадь вычисляется по формуле:

Площадь поверхности
$$\sigma = \iint_D \sqrt{1 + \left(\dfrac{\partial f}{\partial x}\right)^2 + \left(\dfrac{\partial f}{\partial y}\right)^2}\;dx\,dy$$

Множитель под интегралом есть длина нормального вектора к поверхности, проецируемой на плоскость $Oxy$.

8.4 Физические приложения

Пусть плоская фигура $D$ имеет поверхностную плотность $\rho(x,y)$. Тогда:

Физические величины
\begin{aligned} &\text{Масса:} &m &= \iint_D \rho(x,y)\,dA \\[6pt] &\text{Статические моменты:} &M_x &= \iint_D y\,\rho\,dA, \quad M_y = \iint_D x\,\rho\,dA \\[6pt] &\text{Центр масс:} &\bar{x} &= \dfrac{M_y}{m}, \quad \bar{y} = \dfrac{M_x}{m} \\[6pt] &\text{Момент инерции:} &I_O &= \iint_D (x^2 + y^2)\,\rho\,dA \end{aligned}

Величина Формула
Площадь фигуры $S = \iint_D dx\,dy$
Объём тела $V = \iint_D f(x,y)\,dx\,dy$

Тема девятая
Параметрические поверхности. Поверхности вращения. Плоскость трения

9.1 Параметрическое задание поверхности

Поверхность в пространстве может быть задана параметрически: каждая точка поверхности определяется двумя параметрами $u, v$:

Параметрическая форма
$$\mathbf{r}(u,v) = \bigl(x(u,v),\; y(u,v),\; z(u,v)\bigr), \quad (u,v) \in D$$

При фиксированном $u$ или $v$ получаются координатные линии на поверхности.

9.2 Нормаль к поверхности и элемент площади

Вектор нормали к поверхности в точке определяется как векторное произведение касательных векторов:

Нормаль
$$\mathbf{N} = \mathbf{r}'_u \times \mathbf{r}'_v, \qquad |\mathbf{N}| = \sqrt{EG - F^2}$$

где $E = |\mathbf{r}'_u|^2$, $G = |\mathbf{r}'_v|^2$, $F = \mathbf{r}'_u \cdot \mathbf{r}'_v$ — коэффициенты первой квадратичной формы. Элемент площади поверхности:

$$d\sigma = |\mathbf{r}'_u \times \mathbf{r}'_v|\,du\,dv = \sqrt{EG-F^2}\;du\,dv$$

9.3 Поверхности вращения

Поверхность, образованная вращением кривой $y = f(x)$ вокруг оси $Ox$, описывается уравнением:

Поверхность вращения
$$y^2 + z^2 = f^2(x)$$

В параметрической форме: $x = x$, $y = f(x)\cos\varphi$, $z = f(x)\sin\varphi$, где $\varphi \in [0, 2\pi]$. Площадь боковой поверхности тела вращения:

Площадь тела вращения
$$\sigma = 2\pi \int_a^b |f(x)| \sqrt{1 + f'^2(x)}\,dx$$

9.4 Касательная плоскость и нормаль

Уравнение касательной плоскости к поверхности $F(x,y,z) = 0$ в точке $(x_0,y_0,z_0)$:

Касательная плоскость
$$F'_x(x-x_0) + F'_y(y-y_0) + F'_z(z-z_0) = 0$$

Уравнение нормали к поверхности в той же точке:

$$\dfrac{x-x_0}{F'_x} = \dfrac{y-y_0}{F'_y} = \dfrac{z-z_0}{F'_z}$$

Понятие Формула
Элемент площади $d\sigma = \sqrt{EG-F^2}\;du\,dv$
Площадь тела вращения $\sigma = 2\pi\int_a^b|f|\sqrt{1+f'^2}\,dx$
Касательная плоскость $F'_x\Delta x + F'_y\Delta y + F'_z\Delta z = 0$

Тема десятая
Теорема о дивергенции. Скалярное и векторное поле. Градиент. Поток. Теорема Остроградского

10.1 Скалярное поле. Градиент

Скалярным полем называется функция $u = u(x,y,z)$, определённая в некоторой области пространства. Основная характеристика скалярного поля — его градиент:

Градиент
$$\mathrm{grad}\,u = \nabla u = \left(\dfrac{\partial u}{\partial x},\,\dfrac{\partial u}{\partial y},\,\dfrac{\partial u}{\partial z}\right)$$

Градиент указывает направление наиболее быстрого возрастания поля; его модуль равен скорости роста в этом направлении.

10.2 Векторное поле. Дивергенция и ротор

Векторным полем называется вектор-функция $\mathbf{F}(x,y,z) = (P, Q, R)$. Основные дифференциальные операторы над векторным полем:

Дивергенция
$$\mathrm{div}\,\mathbf{F} = \nabla \cdot \mathbf{F} = \dfrac{\partial P}{\partial x} + \dfrac{\partial Q}{\partial y} + \dfrac{\partial R}{\partial z}$$
Ротор
$$\mathrm{rot}\,\mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \partial_x & \partial_y & \partial_z \\ P & Q & R \end{vmatrix}$$

Физический смысл: дивергенция характеризует «мощность источников» поля в точке; ротор описывает вращательный характер поля.

10.3 Поток векторного поля

Потоком векторного поля $\mathbf{F}$ через поверхность $\Sigma$ с нормалью $\mathbf{n}$ называется поверхностный интеграл второго рода:

Поток
$$\Pi = \iint_\Sigma \mathbf{F}\cdot\mathbf{n}\,d\sigma = \iint_\Sigma (P\,dy\,dz + Q\,dx\,dz + R\,dx\,dy)$$

10.4 Теорема Остроградского–Гаусса

Поток векторного поля через замкнутую поверхность $\Sigma = \partial V$ (граница области $V$) равен тройному интегралу от дивергенции по объёму:

Теорема Остроградского
$$\oiint_{\partial V} \mathbf{F}\cdot\mathbf{n}\,d\sigma = \iiint_V \mathrm{div}\,\mathbf{F}\;dV$$

Теорема позволяет заменить сложное поверхностное интегрирование объёмным, если дивергенция поля проста, и наоборот.

Тема одиннадцатая
Потенциальные и соленоидальные векторные поля

11.1 Потенциальное поле

Векторное поле $\mathbf{F}$ называется потенциальным (консервативным), если существует скалярная функция $u(x,y,z)$ — потенциал, — такая что:

Определение потенциала
$$\mathbf{F} = \nabla u = \mathrm{grad}\,u, \quad \text{т.е.} \quad P = \dfrac{\partial u}{\partial x},\quad Q = \dfrac{\partial u}{\partial y},\quad R = \dfrac{\partial u}{\partial z}$$

11.2 Условие потенциальности

Поле $\mathbf{F} = (P, Q, R)$ является потенциальным тогда и только тогда, когда его ротор тождественно равен нулю:

Критерий
$$\mathrm{rot}\,\mathbf{F} = \mathbf{0} \iff \dfrac{\partial R}{\partial y} = \dfrac{\partial Q}{\partial z},\quad \dfrac{\partial P}{\partial z} = \dfrac{\partial R}{\partial x},\quad \dfrac{\partial Q}{\partial x} = \dfrac{\partial P}{\partial y}$$
Следствие. В потенциальном поле криволинейный интеграл не зависит от пути интегрирования и определяется лишь начальной и конечной точками: $$\int_{AB} \mathbf{F}\cdot d\mathbf{r} = u(B) - u(A)$$

11.3 Нахождение потенциала

Потенциал $u$ восстанавливается из системы уравнений $u'_x = P$, $u'_y = Q$, $u'_z = R$ последовательным интегрированием с определением произвольных функций из оставшихся условий.

11.4 Соленоидальное поле

Векторное поле $\mathbf{F}$ называется соленоидальным (или бездивергентным), если его дивергенция тождественно равна нулю:

Определение
$$\mathrm{div}\,\mathbf{F} = 0 \iff \dfrac{\partial P}{\partial x} + \dfrac{\partial Q}{\partial y} + \dfrac{\partial R}{\partial z} = 0$$

По теореме Остроградского это означает, что поток соленоидального поля через любую замкнутую поверхность равен нулю: поле «не имеет источников». Типичный пример — поле несжимаемой жидкости или магнитное поле.

11.5 Связь между классами полей

Любое потенциальное поле является безвихревым: $\mathrm{rot}(\nabla u) = \mathbf{0}$. Любое соленоидальное поле может быть представлено как ротор некоторого поля: $\mathbf{F} = \mathrm{rot}\,\mathbf{A}$ (векторный потенциал).
